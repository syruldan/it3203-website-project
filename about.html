Project Overview & Methodology

This website was developed as a comprehensive class project designed to demonstrate practical proficiency in modern web development standards. The primary goal was to build a multi-page, educational platform focusing on Search Engine Optimization (SEO) while adhering to strict mobile-first design principles.

Throughout the development process, a strong emphasis was placed on user experience (UX) and clean, maintainable code. The site architecture utilizes modern HTML5 semantic structuring—such as <nav>, <header>, and <main> tags—alongside advanced CSS techniques like CSS Grid and Flexbox. This ensures that the layout fluidly adapts across all devices, from desktop monitors to mobile smartphone screens, without losing readability.

Furthermore, the interactive assessment module on the Quiz page highlights core JavaScript programming skills. It features dynamic Document Object Model (DOM) manipulation, custom event handling, and conditional logic to grade various input types (text, radio buttons, and checkboxes) entirely on the client side, without relying on any external libraries or frameworks.

Academic Focus

As a student passionate about Information Technology, this assignment served as a hands-on application of theoretical concepts, bridging the gap between UI/UX design fundamentals and technical execution.
